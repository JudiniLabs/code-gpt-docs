# 关于 CodeGPT

### 欢迎来到我们关于在 Visual Studio Code 中利用 CodeGPT 的教程！

CodeGPT 是一款强大的扩展，利用大型语言模型（LLMs）的能力，通过人工智能来提升您的编程任务。

:::info
这个工具不仅允许您与工作环境动态交互，还可以创建AI代理，丰富LLMs的上下文，并将您连接到仓库、报告和业务的其他关键领域。
:::
